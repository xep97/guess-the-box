# Guess the Box

A minimalist, addictive web-based game where logic meets luck. Can you find the right box, or will you be left guessing?

---

## How to Play

The rules are simple but the challenge is real:  
A series of boxes are presented on the screen. Only one box contains the prize and gets you to next lever. Then you try guessing as many correct boxes as possible in a row. If you fail to find the price, the count resets to 0 again. 

---

## Features

* **Responsive:** Play on your phone, tablet, or desktop.
* **Dynamic Logic:** Randomly generated target locations for infinite replayability.

## Tech Stack

Project is made in standard HTML, CSS and JavaScript.
